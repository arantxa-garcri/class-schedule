Calculadora de sesiones efectivas

apoyo para planeación

Calcula horas reales de clase y verifica el margen DGIRE de ±10% frente a las horas del programa.

Datos del cálculo

Avanza por los pasos: asignatura, grupo y horario, calendario y días sin clase.

1 Asignatura
Revisa y confirma las horas del programa antes de calcular. Este dato es indispensable para comparar el margen DGIRE.
Esta opción permite capturar una asignatura de manera temporal únicamente para realizar el cálculo actual de sesiones efectivas y horas disponibles. La información ingresada no se guardará en el catálogo base, no modificará los datos oficiales de las asignaturas y no quedará registrada para futuros cálculos. Para asignaturas incluidas en el plan de estudios, se recomienda utilizar el catálogo base de la aplicación.
Esta asignatura se usará solo para el cálculo actual. No se guardará en el catálogo.
2 Grupo y horario
Agrega al menos un día de clase para este grupo.
Puedes capturar las horas por unidad si deseas estimar cuándo se completa cada una. Este paso es opcional y no sustituye la planeación didáctica.
3 Calendario y días sin clase
Los días sin clase se restan del cálculo de sesiones efectivas.
